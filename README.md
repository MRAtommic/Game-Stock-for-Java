# Game-Stock-for-Java
เกมจำลองการซื้อขายหุ้น พัฒนาด้วยภาษา Java โดยเน้นหลักการ OOP (Object-Oriented Programming)

1.ด้านบนสุดจะแสดงข้อมูลวัน พร้อมกับจำนวนเงิน และจำนวนหุ้น เมื่อผู้เล่นกดไปที่ buttom newday เกมจะทำการข้ามไปวันถัดไปพร้อมกับราคาหุ้นแต่ละตัวที่อัพเดต โดยที่เกมจะให้เงินตั้งแต่ตอนเริ่มเกมมา 1000$ โดยเกมจะให้เวลาเล่น 200 วันถ้าผู้เล่นสามารถเล่นหุ้นได้ครบ 200 วันและสามารถทำเงินได้ 2000$ ผู้เล่นถึงจะชนะแต่ถ้าไม่ถึงผู้เล่นก็จะแพ้
